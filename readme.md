## 简介

Koishi 的中国象棋插件。支持人人/人机对战。

## 使用

1. 安装 `puppeteer` 服务。
2. 设置指令别名。

## Q&A

1. 如何移动棋子？

- 纵线（炮二平五/炮8平5）或字母坐标（a0a1）。
- 详见 [中国象棋着法表示](https://www.xqbase.com/protocol/cchess_move.htm)。
